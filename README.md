### Smart Waste Monitoring System

Computer vision platform for detecting and monitoring illegal garbage dumping in public spaces using CCTV cameras.

The system combines object detection, human activity analysis, and multi-object tracking to identify suspicious waste disposal events and generate actionable reports for municipal authorities.

A dedicated dashboard provides real-time incident monitoring, evidence management, and analytics for authorized personnel.

**YOLOv8 · MediaPipe · ByteTrack · OpenCV · React · Tailwind CSS · Supabase**

---

### Pothole Detection System

Road infrastructure monitoring system that detects potholes from images and video streams using deep learning models.

Designed to support smart-city initiatives and improve road maintenance through automated defect identification.

**YOLO · OpenCV · Python · Deep Learning**
